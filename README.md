# pdfwrm (PDF WATERMARK REMOVER)
A Termux Program to remove watermarks from PDFs

## Installation
Run the following command to install:

```bash
curl https://raw.githubusercontent.com/dipanshu247k-sys/pdfwrm/refs/heads/main/termux-install.sh | bash
```

## Usage
After installation, use the following command to run:

```bash
proot-distro login ubuntu -- /usr/bin/pdfwrm
```
OR 
```bash
pdfwrm
```
